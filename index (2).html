How Importing Works

• Each import adds onto existing stats — scores accumulate weekend over weekend.
• Paste only that weekend's stats each time — don't re-paste previous weekends.

• Duplicate players on multiple fantasy teams are each counted for their respective owner.
• Use Clear All Stats only if you need to start completely over from scratch.

League Rules

League Format

  • Each team drafts a roster of college basketball players before the NCAA Tournament
  • Teams submit a starting lineup for the first weekend (Round of 64 + Round of 32)
  • After Round 1, player positions and lineup structure are locked for the remainder
  • Players accumulate stats throughout all tournament rounds

Stat Categories (5)

  • Points — cumulative points scored across all games
  • Rebounds — cumulative rebounds across all games
  • Assists — cumulative assists across all games
  • Steals — cumulative steals across all games
  • Blocks — cumulative blocks across all games

Important Rules

  • Stats accumulate across every weekend — each import adds to the running total
  • Players stop earning stats once their real-life team is eliminated
  • Lineup decisions lock after the first weekend — no changes allowed
  • Duplicate players on multiple teams are allowed if not resolved in draft
  • Team name is the tiebreaker if total scores are tied

Prize Structure

  • 🥇 1st Place — $50 cash prize
  • 🥈 2nd Place — $25 cash prize
  • 💀 Last Place — Must buy the commissioner a meatball sandwich from Wawa

Overall Standings

  • Total score = sum of fantasy points across all 5 categories
  • Maximum possible score: 50 points (10 pts × 5 categories)
  • Highest total wins the league
  • Standings recalculate instantly after every import
